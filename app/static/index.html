About JudoCoach AI

What this is

JudoCoach AI is a governed research agent built with LlamaIndex and FastAPI. It retrieves evidence from a private knowledge base, uses a bounded set of tools, and requires explicit human approval before persisting any report.

How approval works

Every question first runs as a draft with no ability to save. Only after you click "Approve & Save" does the agent run again with the save capability present — the tool itself is absent beforehand, not just discouraged by instruction.

Limitations

Small, single-domain knowledge base. No image or video understanding. Retrieval uses a single chunking configuration. See the project README for the full list.
